Design Patterns - Simulasi Interaktif

Demonstrasi pola desain dalam OOP menggunakan contoh real-world dari sistem transaksi retail. Pelajari bagaimana setiap pattern memecahkan masalah yang berbeda dalam desain software.

🎯 Strategy Pattern

Mengubah algoritma diskon secara dinamis saat runtime berdasarkan tipe pelanggan

👁️ Observer Pattern

Notifikasi real-time ke multiple observer saat stok produk berubah

🏭 Factory Pattern

Pembuatan object dokumen (Invoice, Kwitansi) tanpa coupling

📚 Tentang Simulator Ini

Aplikasi ini mengimplementasikan konsep-konsep dari file samples/design-patterns/DiscountStrategy.java dalam bentuk web interaktif. Anda dapat melihat bagaimana design patterns bekerja dalam kehidupan nyata melalui simulasi sistem transaksi belanja.
