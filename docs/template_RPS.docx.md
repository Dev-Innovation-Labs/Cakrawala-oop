| **Deskripsi Singkat MK** | Mata kuliah ini memperkenalkan mahasiswa pada konsep, metode, dan alat dasar dalam sains data. Cakupan materi meliputi seluruh siklus data, mulai dari pengumpulan, pembersihan, transformasi, analisis, hingga visualisasi. Mahasiswa akan mempelajari penerapan metode statistik dasar, pembangunan alur pemrosesan data sederhana, serta interpretasi hasil analisis untuk mendukung pengambilan keputusan. Selain itu, mata kuliah ini menekankan pada evaluasi kualitas dan relevansi sumber data, serta aspek etika dan privasi dalam penggunaan data. Pada akhir mata kuliah, mahasiswa diharapkan mampu menerapkan teknik-teknik dasar sains data pada permasalahan nyata dan menyampaikan temuan secara efektif. |  |  |  |  |  |  |  |  |
