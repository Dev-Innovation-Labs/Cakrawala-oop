CPL-PRODI yang dibebankan pada MK

Unsur Sikap

CPL1

Bertakwa kepada Tuhan Yang Maha Esa, menjunjung tinggi nilai kemanusiaan, serta menjunjung tinggi etika akademik dan profesional dalam praktik sains data.

Unsur Pengetahuan

CPL2

Mampu menguasai dan menerapkan konsep matematika, statistika, machine learning, dan teknologi data science berstandar global untuk memecahkan permasalahan kompleks dan memimpin transformasi digital pada domain spesifik di Indonesia.

Unsur Keterampilan Umum

CPL3

Mampu mengidentifikasi, memformulasikan, dan menyelesaikan tantangan transformasi digital yang kompleks melalui pendekatan data-driven leadership dan solusi sains data inovatif yang berkontribusi pada kemajuan Indonesia sebagai negara berbasis data.

CPL4

Mampu menerapkan pemikiran logis, kritis, sistematis, dan inovatif dalam memimpin pengembangan solusi data science breakthrough yang dapat mengakselerasi transformasi digital dan daya saing Indonesia di kancah global.

CPL5

Mampu menyusun dan mempublikasikan karya ilmiah berkualitas internasional dalam bidang sains data, serta mengkomunikasikan hasil penelitian dan inovasi kepada komunitas global sesuai etika akademik dan standar publikasi internasional.

CPL6

Mampu melakukan evaluasi diri dan memimpin pengembangan tim dalam pembelajaran berkelanjutan teknologi data science terdepan untuk mempertahankan keunggulan kompetitif sebagai pemimpin transformasi digital masa depan.

Unsur Keterampilan Khusus

CPL7

Mampu memimpin proyek sains data strategis mulai dari identifikasi peluang bisnis berbasis data, pengumpulan dan analisis big data, hingga implementasi model data science untuk mendukung pengambilan keputusan level eksekutif dalam transformasi digital organisasi.

CPL8

Mampu merancang dan mengimplementasikan arsitektur data modern berstandar global (database, data warehouse, cloud platforms) dengan menerapkan prinsip data governance dan keamanan data untuk mendukung ekosistem data nasional yang sustainable.

CPL9

Mampu mengevaluasi, mengadopsi, dan memimpin implementasi teknologi dan metodologi data science terdepan sesuai best practices global untuk menghasilkan solusi yang scalable dan mempercepat transformasi digital Indonesia.

CPL10

Mampu menguasai dan memimpin implementasi prinsip-prinsip data governance, cybersecurity, privacy protection, dan ethical AI berstandar global dalam pengembangan ekosistem data nasional yang aman, bertanggung jawab, dan berkelanjutan.

Capaian Pembelajaran Mata Kuliah (CPMK)

CPMK-1

Mampu menjelaskan dan menerapkan konsep dasar sains data (termasuk data, informasi, statistik dasar, dan machine learning) untuk memecahkan permasalahan nyata yang relevan di berbagai bidang. (CPL2, CPL3, CPL7)

CPMK-2

Mampu menggunakan metode statistika dasar (descriptive statistics, probability, inferential analysis) dalam menganalisis dataset sederhana dan menafsirkan hasilnya dengan tepat. (CPL2, CPL4, CPL8)

CPMK-3

Mampu membangun pipeline pengolahan data sederhana, mulai dari data collection, data cleaning, data wrangling, hingga menghasilkan dataset siap analisis.. (CPL2, CPL6, CPL9)

CPMK-4

Mampu menginterpretasikan hasil analisis data dalam bentuk visualisasi dan laporan singkat untuk mendukung proses pengambilan keputusan berbasis data..(CPL3, CPL4, CPL7)

CPMK-5

Mahasiswa mampu mengevaluasi kualitas dan relevansi sumber data (misalnya data publik, data organisasi, atau hasil survei) serta mempertimbangkan aspek etika dalam penggunaannya. (CPL5, CPL10)

 Kemampuan Akhir  Tiap Tahapan  Belajar ( Sub-CPMK)

Sub-CPMK 1

Mampu menjelaskan konsep dasar data, informasi, dan perbedaan keduanya serta kaitannya dengan proses analisis data. (CPMK-1, C2)

Sub-CPMK2

Mampu menerapkan metode statistika dasar (misalnya mean, median, standar deviasi, uji hipotesis sederhana) pada dataset nyata untuk menemukan pola. (CPMK-2, C3)

Sub-CPMK3

Mampu membangun pipeline pengolahan data sederhana mulai dari data collection → cleaning → wrangling → penyimpanan, dengan tools seperti Python (Pandas) atau SQL (CPMK3, C4)

Sub-CPMK4

Mampu membuat visualisasi data dasar (bar chart, line chart, pie chart, histogram) menggunakan tools seperti Excel, Tableau, atau Python untuk menyajikan hasil analisis. (CPMK-4, C3)

Sub-CPMK5

Mampu menginterpretasikan hasil analisis data dan menghubungkannya dengan konteks permasalahan nyata untuk mendukung pengambilan keputusan.(CPMK-4, C4)

Sub-CPMK6

Mampu mengevaluasi sumber data dari aspek kualitas (akurasi, kelengkapan, konsistensi) dan relevansi (kesesuaian dengan tujuan analisis. (CPMK-5, C5)

Sub-CPMK7

Mampu mengidentifikasi isu etika dan privasi dalam penggunaan data serta memberikan rekomendasi praktik yang sesuai dengan prinsip data governance. (CPMK-5, C5)

Deskripsi Singkat MK

Mata kuliah ini memperkenalkan mahasiswa pada konsep, metode, dan alat dasar dalam sains data. Cakupan materi meliputi seluruh siklus data, mulai dari pengumpulan, pembersihan, transformasi, analisis, hingga visualisasi. Mahasiswa akan mempelajari penerapan metode statistik dasar, pembangunan alur pemrosesan data sederhana, serta interpretasi hasil analisis untuk mendukung pengambilan keputusan. Selain itu, mata kuliah ini menekankan pada evaluasi kualitas dan relevansi sumber data, serta aspek etika dan privasi dalam penggunaan data. Pada akhir mata kuliah, mahasiswa diharapkan mampu menerapkan teknik-teknik dasar sains data pada permasalahan nyata dan menyampaikan temuan secara efektif.
