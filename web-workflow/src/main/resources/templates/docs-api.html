5️⃣ Jalankan Transisi Workflow

POST /api/leave/{id}/transisi

Request Body (JSON):
{
  "transisi": "ajukan",
  "alasanTolak": null  // Optional, hanya untuk transisi "tolak"
}

Transisi Tersedia:
- "ajukan"        : DRAFT → DIAJUKAN
- "approve_atasan": DIAJUKAN → DISETUJUI_ATASAN
- "approve_hr"    : DISETUJUI_ATASAN → DISETUJUI_HR
- "tolak"         : DIAJUKAN → DITOLAK (set alasanTolak)
- "batalkan"      : DRAFT → DIBATALKAN

Response (200 OK):
{
  "sukses": true,
  "pesan": "DRAFT → Diajukan",
  "data": {
    "id": "a1b2c3d4-e5f6-4g7h-i8j9-k0l1m2n3o4p5",
    "nama": "Andi Wijaya",
    "status": "DIAJUKAN",
    "statusLabel": "Diajukan",
    "sisaCuti": 12,
    ...
  }
}

Response (400 Bad Request):
{
  "sukses": false,
  "pesan": "Guard gagal: Sisa cuti tidak mencukupi"
}

📋 Contoh Lengkap dengan cURL

# 1. Buat pengajuan baru
curl -X POST http://localhost:8080/api/leave/create \
  -H "Content-Type: application/json" \
  -d '{
    "nama": "Andi Wijaya",
    "email": "andi@example.com",
    "mulai": "2026-05-10",
    "selesai": "2026-05-15",
    "sisaCuti": 12
  }'

# Response akan berisi ID pengajuan, misal: a1b2c3d4-...

# 2. Ajukan pengajuan ke atasan
curl -X POST http://localhost:8080/api/leave/a1b2c3d4-.../transisi \
  -H "Content-Type: application/json" \
  -d '{"transisi": "ajukan"}'

# 3. Setujui oleh atasan
curl -X POST http://localhost:8080/api/leave/a1b2c3d4-.../transisi \
  -H "Content-Type: application/json" \
  -d '{"transisi": "approve_atasan"}'

# 4. Setujui oleh HR (sisa cuti otomatis dikurangi)
curl -X POST http://localhost:8080/api/leave/a1b2c3d4-.../transisi \
  -H "Content-Type: application/json" \
  -d '{"transisi": "approve_hr"}'

# 5. Lihat detail final
curl http://localhost:8080/api/leave/a1b2c3d4-...

⚠️ Error Handling

404 Not Found:
{
  "error": "Request tidak ditemukan"
}

400 Bad Request (Guard gagal):
{
  "sukses": false,
  "pesan": "Guard gagal: Pengajuan cuti harus minimal H-3 (3 hari sebelumnya)"
}

400 Bad Request (Transisi tidak valid):
{
  "sukses": false,
  "pesan": "Transisi 'invalid' tidak valid dari status DRAFT"
}

400 Bad Request (Data tidak valid):
{
  "error": "For input string: \"abc\""
}

🔍 Status Codes

Code Arti Keterangan
200 OK Request berhasil
201 Created Resource baru berhasil dibuat
400 Bad Request Data tidak valid atau guard gagal
404 Not Found Resource tidak ditemukan
500 Internal Server Error Error di server
